## 4. FAQ PAGE (`/faq`)

**Purpose:** Comprehensive FAQ with search and category filtering

```tsx
import React, { useState, useMemo } from 'react';
import { BackButton } from '../components/BackButton';
import { Input } from '../components/ui/input';
import { Tabs, TabsList, TabsTrigger, TabsContent } from '../components/ui/tabs';
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent,
} from '../components/ui/accordion';
import { Button } from '../components/ui/button';
import { Search, Mail, MessageSquare } from 'lucide-react';

export default function FAQPage() {
  const [searchQuery, setSearchQuery] = useState('');
  const [activeCategory, setActiveCategory] = useState('general');

  const faqCategories = {
    general: {
      title: 'General Questions',
      faqs: [
        {
          question: 'What is FlashFusion?',
          answer: 'FlashFusion is an AI-powered development platform that transforms ideas into production-ready applications. We combine 60+ AI tools, instant deployment, and multi-agent orchestration to help developers build 10x faster.'
        },
        {
          question: 'How does FlashFusion work?',
          answer: 'FlashFusion uses advanced AI models (GPT-4, Claude, Gemini) to understand your requirements and generate complete applications including frontend, backend, database schema, and deployment configuration. Our multi-agent system orchestrates different AI specialists to ensure high-quality output.'
        },
        {
          question: 'Do I need coding experience to use FlashFusion?',
          answer: 'No! FlashFusion is designed for everyone from non-technical founders to senior developers. You can start with natural language descriptions and let AI generate the code, or dive deep and customize every aspect of your application.'
        },
        {
          question: 'What types of applications can I build?',
          answer: 'You can build virtually any web application: SaaS products, e-commerce stores, mobile apps, dashboards, APIs, landing pages, and more. FlashFusion supports full-stack development with modern frameworks like React, Next.js, Node.js, and Python.'
        },
      ]
    },
    pricing: {
      title: 'Pricing & Billing',
      faqs: [
        {
          question: 'Is there a free tier?',
          answer: 'Yes! Our Free tier includes 5 projects, basic AI tools, and community support. Perfect for trying out FlashFusion and building personal projects.'
        },
        {
          question: 'What payment methods do you accept?',
          answer: 'We accept all major credit cards (Visa, Mastercard, American Express, Discover) and PayPal. Enterprise customers can also pay via invoice with NET 30 terms.'
        },
        {
          question: 'Can I cancel my subscription anytime?',
          answer: 'Absolutely! You can cancel your subscription at any time from your account settings. Your access will continue until the end of your current billing period. No cancellation fees, no questions asked.'
        },
        {
          question: 'Do you offer refunds?',
          answer: 'Yes, we offer a 30-day money-back guarantee on all paid plans. If you\'re not satisfied within the first 30 days, contact support for a full refund.'
        },
        {
          question: 'What happens if I exceed my plan limits?',
          answer: 'We\'ll notify you when you approach your limits. You can either upgrade your plan or additional usage will be billed at standard overage rates. We never cut off access without warning.'
        },
      ]
    },
    technical: {
      title: 'Technical Questions',
      faqs: [
        {
          question: 'What AI models does FlashFusion use?',
          answer: 'We support 8+ leading AI models including GPT-4, GPT-4 Turbo, Claude 3 Opus, Claude 3 Sonnet, Gemini Pro, Gemini Ultra, and specialized models for specific tasks. You can switch between models based on your needs.'
        },
        {
          question: 'Which deployment platforms are supported?',
          answer: 'FlashFusion supports one-click deployment to Vercel, Netlify, AWS, Google Cloud, Azure, Railway, Render, and DigitalOcean. We handle all configuration and setup automatically.'
        },
        {
          question: 'Can I export my code and self-host?',
          answer: 'Yes! All code generated by FlashFusion is fully exportable. You own 100% of the code and can download it as a complete repository, deploy it anywhere, or continue development locally.'
        },
        {
          question: 'What frameworks and languages are supported?',
          answer: 'Frontend: React, Next.js, Vue, Svelte, Angular. Backend: Node.js, Python (FastAPI/Django), Go, Ruby. Databases: PostgreSQL, MySQL, MongoDB, Redis, Supabase. Mobile: React Native, Flutter.'
        },
        {
          question: 'Is my code and data secure?',
          answer: 'Absolutely. We use bank-level encryption (AES-256), SOC 2 Type II compliance, GDPR adherence, and regular security audits. Your code is never used to train AI models without explicit permission.'
        },
      ]
    },
    features: {
      title: 'Features & Capabilities',
      faqs: [
        {
          question: 'What is multi-agent orchestration?',
          answer: 'Our multi-agent system uses specialized AI agents that work together: a Frontend Agent handles UI, a Backend Agent manages server logic, a Database Agent designs schemas, and a DevOps Agent handles deployment. They collaborate to build complete applications.'
        },
        {
          question: 'Can I collaborate with my team?',
          answer: 'Yes! Team and Enterprise plans include real-time collaboration, code review workflows, version control integration, and team management features. Multiple developers can work on the same project simultaneously.'
        },
        {
          question: 'Does FlashFusion integrate with GitHub?',
          answer: 'Yes! FlashFusion seamlessly integrates with GitHub for repository management, pull requests, and CI/CD workflows. You can also connect to GitLab and Bitbucket.'
        },
        {
          question: 'Are there pre-built templates?',
          answer: 'Yes! We offer 50+ production-ready templates across categories: SaaS, e-commerce, portfolios, dashboards, landing pages, and more. Templates are fully customizable and follow best practices.'
        },
      ]
    },
  };

  const filteredFAQs = useMemo(() => {
    if (!searchQuery) return faqCategories[activeCategory].faqs;
    
    const query = searchQuery.toLowerCase();
    return faqCategories[activeCategory].faqs.filter(faq =>
      faq.question.toLowerCase().includes(query) ||
      faq.answer.toLowerCase().includes(query)
    );
  }, [searchQuery, activeCategory]);

  return (
    <div className="min-h-screen bg-background text-foreground">
      <div className="container mx-auto px-4 py-12 max-w-4xl">
        <BackButton />

        <div className="mb-12 ff-fade-in-up">
          <h1 className="text-4xl md:text-6xl font-bold mb-4">
            <span className="ff-text-gradient">Frequently Asked Questions</span>
          </h1>
          <p className="text-xl text-muted-foreground max-w-2xl">
            Find answers to common questions about FlashFusion
          </p>
        </div>

        {/* Search Bar */}
        <div className="relative mb-8">
          <Search className="absolute left-3 top-1/2 transform -translate-y-1/2 h-5 w-5 text-muted-foreground" />
          <Input
            type="search"
            placeholder="Search FAQs..."
            value={searchQuery}
            onChange={(e) => setSearchQuery(e.target.value)}
            className="ff-focus-ring pl-10"
          />
        </div>

        {/* Category Tabs */}
        <Tabs value={activeCategory} onValueChange={setActiveCategory} className="mb-8">
          <TabsList className="grid grid-cols-2 lg:grid-cols-4 gap-2 bg-surface p-2 rounded-lg">
            {Object.entries(faqCategories).map(([key, category]) => (
              <TabsTrigger
                key={key}
                value={key}
                className="data-[state=active]:bg-primary data-[state=active]:text-white"
              >
                {category.title}
              </TabsTrigger>
            ))}
          </TabsList>

          {Object.entries(faqCategories).map(([key]) => (
            <TabsContent key={key} value={key} className="mt-8">
              {filteredFAQs.length > 0 ? (
                <Accordion type="single" collapsible className="space-y-4">
                  {filteredFAQs.map((faq, idx) => (
                    <AccordionItem
                      key={idx}
                      value={`item-${idx}`}
                      className="bg-surface border border-border rounded-lg px-6"
                    >
                      <AccordionTrigger className="text-left hover:no-underline py-4">
                        <span className="font-semibold text-foreground">
                          {faq.question}
                        </span>
                      </AccordionTrigger>
                      <AccordionContent className="text-muted-foreground pb-4">
                        {faq.answer}
                      </AccordionContent>
                    </AccordionItem>
                  ))}
                </Accordion>
              ) : (
                <div className="text-center py-12">
                  <p className="text-muted-foreground mb-4">
                    No FAQs found matching "{searchQuery}"
                  </p>
                  <Button
                    variant="outline"
                    onClick={() => setSearchQuery('')}
                  >
                    Clear Search
                  </Button>
                </div>
              )}
            </TabsContent>
          ))}
        </Tabs>

        {/* Still Have Questions CTA */}
        <div className="mt-16 text-center bg-surface rounded-lg p-8 border border-border">
          <h3 className="text-2xl font-bold mb-4">
            Still have questions?
          </h3>
          <p className="text-muted-foreground mb-6">
            Can't find the answer you're looking for? Our support team is here to help.
          </p>
          <div className="flex flex-col sm:flex-row gap-4 justify-center">
            <Button className="ff-btn-primary" asChild>
              <a href="/contact">
                <Mail className="mr-2 h-5 w-5" />
                Email Support
              </a>
            </Button>
            <Button variant="outline">
              <MessageSquare className="mr-2 h-5 w-5" />
              Live Chat
            </Button>
          </div>
        </div>
      </div>
    </div>
  );
}
```
